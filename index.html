Términos de Uso y Política de Privacidad de RXT Security

Aceptación de los Términos

Al invitar y utilizar RXT Security en tu servidor de Discord, aceptas los términos y condiciones establecidos en este documento. Si no estás de acuerdo con estos términos, te pedimos que no utilices el bot.

Si eres menor de edad, asegúrate de obtener el consentimiento de tus padres o tutores legales antes de usar el bot.

Descripción de los Servicios

RXT Security es un bot de moderación y seguridad diseñado para:

  • Filtrar y eliminar contenido inapropiado.
  • Moderar a los usuarios (advertencias, expulsiones, baneos).
  • Proteger el servidor de spam, comportamientos abusivos y otras actividades no deseadas.
  • Registrar y almacenar temporalmente eventos de moderación para referencia de los administradores.

Política de Privacidad

Información Recopilada

RXT Security no recopila información personal sensible, pero puede recopilar los siguientes datos necesarios para la moderación y el funcionamiento del bot:

  • Datos de Discord: Identificadores de usuario (ID de usuario, nombre de usuario) y cualquier otra información relacionada con tu cuenta de Discord.
  • Mensajes: Mensajes enviados en los canales donde el bot está activo.
  • Acciones de Moderación: Información sobre las acciones realizadas por el bot, como advertencias, expulsiones, baneos, etc.

Uso de los Datos

La información recopilada se usa exclusivamente para los siguientes fines:

  • Moderación: Para realizar tareas de moderación dentro del servidor de Discord, como filtrar contenido y gestionar usuarios.
  • Seguridad: Para mejorar la protección del servidor y prevenir comportamientos abusivos o peligrosos.

Almacenamiento de Datos

Los datos recogidos (como registros de moderación) son almacenados temporalmente y eliminados automáticamente después de un tiempo determinado o cuando el bot deja de estar activo en el servidor.

Compartir Información

RXT Security no compartirá tus datos con terceros, excepto en situaciones excepcionales, como para cumplir con la ley o para prevenir fraudes o abusos.

Responsabilidades del Usuario

Como propietario o administrador de un servidor, eres responsable de garantizar que el uso de RXT Security cumpla con las leyes locales y las políticas de Discord. También eres responsable de informar a los miembros de tu servidor sobre el uso del bot y de sus funcionalidades.

Seguridad de los Datos

RXT Security implementa medidas de seguridad para proteger la información que procesa. Sin embargo, ten en cuenta que no podemos garantizar la seguridad absoluta de los datos, dado que las comunicaciones en línea siempre tienen ciertos riesgos inherentes.

Modificaciones de los Términos

Nos reservamos el derecho de modificar estos Términos de Uso y la Política de Privacidad en cualquier momento. Las actualizaciones se publicarán aquí con la fecha de la última actualización reflejada al principio de este documento.
